# 小R AI助手

小R AI助手 - 您的智能伙伴，随时随地为您提供帮助。这是一个功能强大的AI助手扩展，集成多种实用功能。

## 功能特性

- **AI对话助手**：智能AI助手，随时回答您的问题
- **多AI模型支持**：支持多种AI模型包括DeepseekV3.2、豆包、腾讯元宝、Qwen3、Gemini-2.5等
- **实用技能工具**：
  - AI图片生成：根据描述生成图片
  - 图片OCR识别：从图片中提取文字
  - 翻译助手：多语言翻译功能
  - AI编程助手：代码编写和解释
  - 天气查询：实时天气信息
  - 新闻查询：最新新闻资讯
- **深色模式支持**：自动适配系统的颜色方案
- **个性化设置**：丰富的自定义选项
- **键盘快捷键**：使用 `Ctrl+Shift+S` 快速打开助手

## 安装方法

1. 下载或克隆此仓库
2. 打开 Chrome 并导航到 `chrome://extensions`
3. 在右上角启用"开发者模式"
4. 点击"加载已解压的扩展程序"并选择扩展文件夹
5. 扩展图标将出现在您的工具栏中

## 使用方法

- 点击工具栏中的扩展图标打开弹出窗口，然后点击"打开小R AI助手"
- 或者使用键盘快捷键 `Ctrl+Shift+S` 快速打开助手
- 在助手界面中输入您的问题
- 使用下方的 💡 技能按钮访问各种实用功能
- 在设置中自定义AI模型和其他偏好设置

## 开发说明

此扩展使用 Manifest V3 构建，并使用 Chrome 侧边栏面板 API。

### 构建图标

从 SVG 源文件重新生成图标：

```bash
node build-icons.js
```

这将从源 SVG 生成多种尺寸的 PNG 图标。

## 权限说明

此扩展需要以下权限：
- `sidePanel`：用于显示侧边栏
- `activeTab`：用于与当前标签页交互
- `storage`：用于保存用户偏好设置（此版本尚未实现）
