# 小R AI助手

小R AI助手是一个基于Electron开发的桌面应用程序，集成了多种AI功能，包括对话、图片生成、OCR识别、翻译等。

## 功能特性

- **多AI模型支持**：支持DeepseekV3.2、豆包、腾讯元宝、Qwen3等多种AI模型
- **对话管理**：支持多轮对话、对话历史记录、对话标题自动生成
- **技能模式**：
  - AI图片生成：根据描述生成图片
  - OCR识别：识别图片中的文字
  - 翻译助手：提供翻译功能
- **语音功能**：支持AI语音回复和语音输入
- **主题切换**：支持深色/浅色主题
- **上下文管理**：可设置对话历史上下文数量
- **数据清除**：一键清除所有数据功能
- **新年特效**（v1.0.1）：
  - 雪花特效：营造节日氛围
  - 爆竹动画：可点击触发烟花效果
  - 按钮装饰：增加节日元素
  - 新年祝福：自动显示新年祝福
  - 特效开关：可设置开启或关闭雪花特效

## 开发环境的调试运行

1. 确保系统已安装Node.js
2. 安装依赖：
   ```bash
   npm install
   ```
3. 运行应用：
   ```bash
   npm start
   ```

## 使用说明

### 基本对话
- 在输入框中输入问题，点击"发送"或按回车键发送
- AI助手会根据上下文进行回复

### 技能模式
- 点击对应技能按钮启用相应功能
- 图片生成：输入图片描述，AI会生成相应图片链接
- OCR识别：输入包含图片链接的消息，AI会识别图片文字
- 翻译：AI会自动识别并翻译输入内容

### 设置面板
- 点击设置按钮（⚙️）打开设置面板
- 可设置主题、上下文数量、语音开关、AI模型等

### 对话管理
- 点击"对话列表"查看所有对话
- 支持新建对话、切换对话、重命名对话、删除对话

## 一键清除数据
在设置面板中点击"清除所有数据"按钮可清除所有对话历史和用户设置。

## 停止生成功能
在AI回复过程中，点击"停止"按钮可中断当前回复生成。
